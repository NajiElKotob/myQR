# myQR
* QR code generator for Windows
* [Download it from the Microsoft Store](https://apps.microsoft.com/detail/9N4RZP7H0828?hl=en-us&gl=US&ocid=pdpshare)

```

myQR is a simple and fast QR code generator for Windows.

Type or paste any URL to instantly generate a QR code.
Save your QR codes to history with labels and notes.
Export as PNG, copy to clipboard, or share directly via
WhatsApp and the Windows share sheet.

- Real-time QR code generation
- Save history with labels and notes  
- Export as PNG image
- Copy to clipboard
- Share via WhatsApp
- Windows native share sheet
- 100% offline — no internet required
- No ads, no tracking, no data collection

```
